# DB 문서

Tumo Backend 데이터베이스 문서를 관리하는 디렉터리다.

## 현재 상태

현재 단계명 기준의 DB 문서는 사용하지 않는다.

기존 단계별 스키마 문서는 현재 코드와 맞지 않는 오래된 문서라 삭제했다. 특히 현재는 `stocks`, `holdings`, `refresh_tokens` 등 여러 도메인 테이블이 코드에 존재하므로, 과거 최소 스키마 문서를 유지하면 혼선을 만든다.

## 작성 기준

DB 문서를 다시 작성할 때는 현재 Entity와 실제 마이그레이션 정책을 기준으로 한다.

포함할 내용:

- 테이블 목록
- 주요 컬럼과 타입
- 연관관계
- 인덱스와 제약조건
- 금액/수량/수익률 계산 기준
- `ddl-auto` 또는 마이그레이션 도구 사용 정책
